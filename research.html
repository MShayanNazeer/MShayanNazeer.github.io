Research

My research focuses on developing innovative algorithms and systems for sustainable computing, security, and optimization of AI systems.

Sustainable AI Systems

This research aims to develop energy-efficient algorithms for deep learning models that can run on edge devices with minimal energy consumption. We've developed novel pruning techniques that reduce computational requirements while maintaining accuracy.

Key Findings:

  • Achieved 70% reduction in energy consumption for CNN models
  • Developed adaptive pruning algorithms that adjust to hardware constraints
  • Created a framework for measuring and optimizing carbon footprint of AI deployments

Collaborators: Stanford Green Computing Lab, Microsoft Research

Secure Distributed Computing

This project investigates methods to secure distributed computing environments while maintaining performance. We're developing privacy-preserving machine learning techniques that allow collaborative model training without exposing sensitive data.

Key Findings:

  • Created a secure federated learning protocol with formal privacy guarantees
  • Demonstrated resilience against various adversarial attacks
  • Designed a lightweight encryption scheme suitable for resource-constrained IoT devices

Collaborators: Google AI, Stanford Security Lab

Edge Computing Optimization

This research explores techniques to optimize the deployment and execution of AI models on edge devices. We're developing methods to automatically partition and distribute computation across cloud and edge infrastructure.

Key Findings:

  • Reduced latency by 65% through intelligent workload distribution
  • Created an adaptive framework that responds to network conditions and device capabilities
  • Developed compression techniques specific to edge-cloud communication patterns

Collaborators: Intel Labs, Edge Computing Consortium
